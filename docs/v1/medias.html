الواجهة البرمجيّة للتطبيق (API)

الوسائط المتعدّدة: مفاهيم أساسيّة

يُتيح تطبيق تينه مشاركة الوسائط المتعدّدة (الصور وَ الفيديو و الصوت) فيما بين الأفراد، كما يُتيح للأفراد الإعجاب بهذه الوسائط و كذلك التعليق عليها، و هذه الصفحة خُصّصت لشرح هذه الوظائف بإسهاب.

الإعجاب بأحد الوسائط المتعدّدة

يُمكن للمستخدم الحاليّ إبداء إعجابه بأحد الوسائط التي تمّ مشاركتها معه من خلال إرسال طلب GET إلى الرابط التالي:

و ردّ الخادم سيكون أحد الردود التالية: إمّا 204 أو 403:

الطلب
GET 
{{teenah_api_url}}/medias/{id}/like
  • {id} (مطلوب) رقم أو معرّف الوسائط المتعدّدة.

و ردّ الخادم سيكون أحد الردود التالية: إمّا 204 أو 403:

الردود
204
403 
{
    "message": "Not authroized to use this resource."
}
  • {message} رسالة الخطأ العائدة من الخادم.

التعليق على أحد الوسائط المتعدّدة

يُمكن للمستخدم الحاليّ أن يعلّق على أيٍّ من الوسائط المتعدّدة التي يمكنه الوصول إليها من خلال إرسال طلب POST إلى الرابط التالي:

الطلب
POST 
{{teenah_api_url}}/medias/{id}/comment
  • {id} (مطلوب) رقم أو معرّف الوسائط المتعدّدة.
  • {comment} (مطلوب) (في الرؤوس) نص التعليق.

و ردّ الخادم سيكون أحد الردود التالية: إمّا 204 أو 400 أو 403:

الردود
204
400 
{
    "message": "Bad request."
}
  • {message} رسالة الخطأ العائدة من الخادم.
403 
{
    "message": "Not authroized to use this resource."
}
  • {message} رسالة الخطأ العائدة من الخادم.

الإعجاب بتعليق على الوسائط المتعدّدة

كما يُمكن للمستخدم الحاليّ الإعجاب بتعليق فردٍ من الأفراد على أيٍّ من الوسائط المتعدّدة من خلال إرسال طلب GET إلى الرابط التالي:

الطلب
GET 
{{teenah_api_url}}/medias/{media_id}/comments/{comment_id}/like
  • {member_id} (مطلوب) (في الرابط) رقم أو معرّف الفرد المُراد.
  • {comment_id} (مطلوب) (في الرابط) رقم أو معرّف التعليق.

و ردّ الخادم سيكون أحد الردود التالية: إمّا 204 أو 403:

الردود
204
403 
{
    "message": "Not authroized to use this resource."
}
  • {message} رسالة الخطأ العائدة من الخادم.
