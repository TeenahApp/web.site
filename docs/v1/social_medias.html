الواجهة البرمجيّة للتطبيق (API)

الشبكات الاجتماعيّة: مفاهيم أساسيّة

يُتيح تطبيق تينه للمستخدم إضافة الشبكات الإجتماعيّة التي يريد من خلال إرسال POST إلى الرابط التالي، علماً أن أكواد الردّ للخادم ستكون إمّا 201 أو 400، كما ينبغي أن يضمّن اسم الشبكة الاجتماعيّة باللغة الإنجليزيّة:

الطلب
POST 
{{teenah_api_url}}/socialmedias
  • {name} (مطلوب) اسم الشبكة الاجتماعيّة المُرادة.
  • {account} (مطلوب) معرّف أو حساب الشخص في هذه الشبكة الاجتماعيّة -وليس الرابط-.
الردود
201 
{
    "message": "Social media account has been successfully created."
}
  • {message} رسالة التمام التي تمّ استقبالها من الخادم.
400 
{
    "message": "Bad request."
}
  • {message} رسالة الخطأ التي تمّ استقبالها من الخادم.

أمّا إن أردت تحديث حساب الفرد في أحد الشبكات الاجتماعيّة فبإمكانك ذلك من خلال ارسال طلب PUT إلى الرابط:

الطلب
PUT 
{{teenah_api_url}}/socialmedias/{id}
  • {id} (مطلوب) (في الرابط) رقم أو معرّف الشبكة الاجتماعيّة للفرد.
  • {account} (مطلوب) (في الرؤوس) حساب الفرد في الشبكة الاجتماعيّة.
الردود
200 
{
    "message": "Social media account has been updated successfully."
}
  • {message} رسالة التمام التي تمّ استقبالها من الخادم.
400 
{
    "message": "Bad request."
}
  • {message} رسالة الخطأ التي تمّ استقبالها من الخادم.

في حال الرغبة في حذف شبكة اجتماعيّة للفرد، فإن ذلك يعتبر ممكناً من خلال إرسل طلب DELETE إلى الرابط التالي:

الطلب
DELETE 
{{teenah_api_url}}/socialmedias/{id}
  • {id} (مطلوب) رقم أو معرّف الشبكة الاجتماعيّة للفرد.
الردود
204
400 
{
    "message": "Bad request."
}
  • {message} رسالة الخطأ التي تمّ استقبالها من الخادم.
